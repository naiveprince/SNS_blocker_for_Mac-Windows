# セキュリティポリシー

SNS Access Blocker のセキュリティ上の問題を見つけた場合は、GitHub Issue で報告してください。

報告時には、可能な範囲で次の情報を含めてください。

- OS とバージョン。
- アプリのバージョン、または使用した Release / commit。
- 再現手順。
- 期待した動作。
- 実際の動作。

特に、次のような報告を歓迎します。

- hosts ファイル以外のファイルを意図せず変更する。
- ブロック解除後に hosts ファイルが元に戻らない。
- 予期しない外部通信が発生する。
- 管理者権限まわりの問題。
- 一時ファイルや一時スクリプトの安全性の問題。
- Windows の自動解除タスクが残る問題。

このアプリは hosts ファイルを一時的に編集するため、macOS / Windows の管理者権限を必要とします。管理者権限が関係する不審な挙動や、説明と異なる挙動を見つけた場合も Issue で知らせてください。
