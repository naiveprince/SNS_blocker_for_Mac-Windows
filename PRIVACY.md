# プライバシーポリシー

SNS Access Blocker は、個人情報を収集・保存・送信・共有しません。

このアプリはローカルで動作し、外部サーバーと通信しません。利用状況、ブロックしたサービス、入力した追加ドメインなどを外部へ送信することはありません。

このアプリは、次の情報を読み取りません。

- ブラウザ履歴。
- Cookie。
- パスワード。
- メッセージ。
- 写真。
- 連絡先。
- ドキュメント。

このアプリの主なシステムレベルの変更は、選択されたドメインをブロックするための hosts ファイルの一時的な更新です。ブロック設定の反映や解除のため、DNS キャッシュを更新 / フラッシュする場合があります。

Windows では、自動解除のために一度限りのタスクと解除用スクリプトを作成する場合があります。解除時には、関連するタスクと一時スクリプトの削除を試みます。

また、ブロック状態の表示のため、ユーザーのホームフォルダに `.sns_access_blocker_state.json` を作成 / 削除する場合があります。このファイルはローカルに保存され、外部へ送信されません。
